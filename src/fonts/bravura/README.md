## Sobre

Bravura é uma fonte de notação musical que implementa o padrão SMuFL (Standard Music Font Layout). 
É desenvolvida pela Steinberg e é a fonte padrão usada no Dorico.

## Links

- Repositório: https://github.com/steinbergmedia/bravura

## Uso no Projeto

Estes arquivos são referenciados em `src/assets/fonts/bravura-font.css` e são usados
para renderizar símbolos musicais profissionais em toda a aplicação.
